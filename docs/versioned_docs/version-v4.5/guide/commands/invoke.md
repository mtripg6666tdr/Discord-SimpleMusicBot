# `インボーク`コマンド
指定されたコマンドを実行します。基本的に使用しないでください。

スラッシュコマンドでは、`/invoke`を使用してください。

## 別名
`インボーク`以外にも以下の別名を使用できます。

- invoke

## 使い方
```
invoke <コマンド>
```

## 使用例
```
invoke play 夜に駆ける
```


## 実行に必要な権限
なし

※管理者権限や、サーバーの管理権限、チャンネルの管理権限、および管理者権限を持つユーザーはこの権限を満たしていなくてもいつでもこのコマンドを実行できます。
