# `削除`コマンド
キュー内の指定された位置の曲を削除します。

スラッシュコマンドでは、`/remove`を使用してください。

## 別名
`削除`以外にも以下の別名を使用できます。

- 消去
- remove
- rm
- del
- delete

## 使い方
```
削除 <削除する位置>
```

## 使用例
```
rm 5
```


## 実行に必要な権限
なし

※管理者権限や、サーバーの管理権限、チャンネルの管理権限、および管理者権限を持つユーザーはこの権限を満たしていなくてもいつでもこのコマンドを実行できます。
