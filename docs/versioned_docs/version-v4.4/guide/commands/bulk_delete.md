# `バルク削除`コマンド
ボットが送信したメッセージを一括削除します。過去1000件のメッセージを遡って検索します。

スラッシュコマンドでは、`/bulk_delete`を使用してください。

## 別名
`バルク削除`以外にも以下の別名を使用できます。

- bulk_delete
- bulk-delete
- bulkdelete




## 実行に必要な権限
メッセージの管理権限を持っていること

※管理者権限や、サーバーの管理権限、チャンネルの管理権限、および管理者権限を持つユーザーはこの権限を満たしていなくてもいつでもこのコマンドを実行できます。
