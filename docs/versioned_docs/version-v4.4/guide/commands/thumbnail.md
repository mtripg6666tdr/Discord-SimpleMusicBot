# `サムネイル`コマンド
現在再生中のサムネイルを表示します。検索パネルが開いていて検索パネル中の番号が指定された場合にはその曲のサムネイルを表示します。

スラッシュコマンドでは、`/thumbnail`を使用してください。

## 別名
`サムネイル`以外にも以下の別名を使用できます。

- サムネ
- thumbnail
- thumb
- t

## 使い方
```
サムネイル [検索パネル中のインデックス]
```

## 使用例
```
サムネイル 5
```


## 実行に必要な権限
なし

※管理者権限や、サーバーの管理権限、チャンネルの管理権限、および管理者権限を持つユーザーはこの権限を満たしていなくてもいつでもこのコマンドを実行できます。
